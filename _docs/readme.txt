/**
 * Pickles Framework ドキュメント
 */

このディレクトリには、Pickles Framework の関連ドキュメントを格納しています。

■PxFW_1x_managementManual.odt - サイト構築管理マニュアル

サイト全体の管理者(=多くの場合テーマの実装者)向けのマニュアルです。
Pickles Framework の機能や仕組み、全体の規則などについて記述しています。
通常、管理者は1サイトにつき1人程度の少人数となるでしょう。
少人数チームで、大きなウェブサイトを効率よく管理するための仕組み、機能、
ルールについて書かれています。


■PxFW_1x_contentsManual.odt - コンテンツ制作マニュアル

コンテンツを制作する実装者向けのマニュアルです。
コンテンツ制作は通常、複数の担当者が同時進行で作業を進めますが、
それぞれの担当範囲は1ページ単位と狭く、全体を見通せる環境では
作業していません。
そういう前提でも、全体の管理が煩雑にならないよう、
コンテンツ制作者は共通の安全なルールで実装を進める必要があります。


---
これらドキュメントに書かれた内容は、
Pickles Framework が定義する標準的で推奨されたルールです。
そのため、プロジェクトやチームによっては、
一部あるいは大部分が使いにくい内容になっている可能性もあるでしょう。

そのような場合は、これらのドキュメントをテンプレートとして、
それぞれの個別の要件にあわせて修正して使用することもできます。
